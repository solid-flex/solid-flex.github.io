SOLID FLEX
Getting Started How to Use Reference Github

Ultra Light-weight Grid System Library using CSS Flex

Solid Flex is a Flex-based UI library that fully controls grid layouts in a web page. It is very easy to use and does not require CSS Flex knowledge.

Get Started

Getting Started

Solid Flex Usage is very simple.
To use, simply insert the original Solid Flex css file into the head tag of your html file. 

<link rel="stylesheet" type="text/css" href="path/to/solid-flex.min.css">

Aafter that, you can use responsive attributes whenever you want.

Example: 
                
                    <div class="r">
                        <div xl="4" lg="6" md="8" sm="12" xs="24">1</div>
                        <div xl="4" lg="6" md="8" sm="12" xs="24">2</div>
                        <div xl="4" lg="6" md="8" sm="12" xs="24">3</div>
                    </div>

How to Use

Every row should have a parent element with class="r".
In that element you can put your columns directly.
Solid Flex use custom HTML attributes to handle grid columns in defferent device sizes. It is important to know that the Solid Flex consists of 24 units. For example, When you set value of 12 to each size it equals to 50%.

Size Name X-Large Size (xl) Large Size (lg) Medium Size (md) Small Size (sm) Extra-Small Size (xs)
Device Width Limit >1200 <1200 <992 <768 <576

Change your browser width to see responsive effect of Solid Flex.

Desired Column Numbers:

When you need to put all columns in one row, you should use rn class for parent instead of r class.
In this case Solid Flex will put all child elements of that parent into one row automatically. Also row width will be evenly divided between columns.
It can be very useful for special grid systems (like one-fifth, One-seventh and etc.)

Horizontally Justify of Elements:

If you want to justify elements in a row you can use Solid Flex custom attributs.
For horizontal justify you can use below attribues :

Attribute Equals to What does it do?
sb space-between

make space between columns evenly.
First and last column will stick to row sides.

sa space-around

make space between columns evenly.
Left and right margin of each element are equal.

se space-evenly

make space between columns evenly.
Space between first and last column with row sides are equal to space between other elements.

end flex-end elements will stick to right side of row
start flex-start elements will stick to left side of row

Vertically Alignment of Elements:

If you want to align your elements vertically, you can use Solid Flex custom attributes too.

Attribute Equals to What does it do?
a-start flex-start

Stick elements to top of the row.

a-end flex-end

Stick elements to bottom of the row.

a-center center

Align element in center of the row (verticaly)

stretch stretch

Stretch elements to fill height of the row.
